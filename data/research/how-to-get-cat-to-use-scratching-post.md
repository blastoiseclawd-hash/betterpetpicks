## Key Patterns from Research

### 1. Demonstrate scratching (mimicry technique)
Multiple owners independently confirm that getting on hands and knees and scratching the post while the cat watches triggers the cat to try it. Cats learn by observing. This is the most commonly upvoted advice across all 3 threads.

### 2. Horizontal vs vertical is a real preference divide
Some cats exclusively prefer horizontal/flat surfaces (carpet, rugs). Offering only vertical posts to a horizontal-preference cat guarantees failure. Watch where the cat currently scratches — carpet means horizontal, couch arm means vertical. Many respondents recommend having both types available.

### 3. Stability is non-negotiable
"My cat has a cat tree that has really fat posts and he likes to run up to it and sharpen his claws viciously." vs "My cats don't like posts if they are tippy." A wobbly post fails instantly. Wide base, heavy base, wall-mounting — whatever makes it solid.

### 4. Wand toy play drives initial engagement
Using a wand toy around the post gets the cat's claws physically onto the surface during play. Once claws contact sisal or cardboard, scratching instinct often takes over. This bypasses the cat's indifference to a static object.

### 5. Catnip spray works but not for all cats
~30% of cats don't respond to catnip (genetic). For those, silvervine is the alternative. Homemade catnip spray: brew dried catnip as tea, strain, put in spray bottle. Daily application keeps interest high.

### 6. Match the texture the cat already scratches
If the cat scratches carpet, give it carpet-textured or cardboard flat scratchers. If it scratches wood, try bark-covered posts. Fighting the cat's texture preference loses every time.

### 7. Previous cat scent on old posts
When replacing a deceased cat's post for a new cat, the old scent can be a deterrent or neutral. If the new cat avoids the old post, replace it. If it doesn't mind, add catnip to renew interest.

### 8. Some cats never use posts
Honest reality from multiple owners: some cats — especially older cats set in their ways — never fully transition. For these, alternatives (outdoor rugs, sacrificial furniture, carpet samples) minimize damage rather than eliminating it.
