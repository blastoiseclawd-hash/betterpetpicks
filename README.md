# BetterPetPicks

Practical cat-care buying guides covering litter, fountains, cat gear, pet-hair cleanup, and dog beds. Revenue from affiliate commissions; recommendations are customer-first.

## Canonical docs
- `AGENTS.md` — governance, rules, current phase, editorial desks
- `data/state.json` — phase state ledger (source of truth for current phase)
- `data/avatar-research.md` — buyer personas
- `data/guide-opportunities.md` — content queue + linkable assets
- `data/linkable-asset-candidates.md` — backlink-earning targets

## Commands
- `npm run dev` — local dev server
- `npm run build` — production build
- `npm run lint` — lint
- From repo root: `npm run audit:site -- --site betterpetpicks`
