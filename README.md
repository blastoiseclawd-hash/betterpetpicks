# BetterPetPicks

BetterPetPicks is the cat-care launch site in the affiliate marketing portfolio. The current foundation pass sets up the brand, trust pages, desk framework, and pre-content infrastructure for the first manual publishing wave.

## Current foundation

- Next.js App Router publishing stack
- BetterPetPicks brand config in `src/config/site.ts`
- Desk framework in `src/data/authors.ts`
- Empty product catalog and keyword inventory for later phases
- Route-aware sitemap plus metadata/schema helpers
- Trust pages, disclosure patterns, and FTC copy wiring

## Next phase priorities

- Replace `data/avatar-research.md` with source-backed buyer research
- Validate or revise the desk framework in `src/data/authors.ts`
- Build `src/data/keyword-map.ts` from research-backed priorities
- Promote the first manual launch pages into `data/guide-opportunities.md`
- Add the first content routes under `src/app`

## Brand constants

The launch foundation is built around these values:

- `BetterPetPicks`
- `betterpetpicks.com`
- `Practical cat-care buying guides for cleaner, calmer homes.`
- `BetterPetPicks publishes practical, independent buying guides for cat litter, odor control, water fountains, and the products that help cat owners keep real homes running smoothly.`
